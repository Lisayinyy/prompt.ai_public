# prompt.ai Competition Demo

## One-line positioning
prompt.ai helps users turn rough intent into a structured AI task, match it with the right model, and launch it directly into AI chat.

## 30-second version
prompt.ai is no longer just a prompt optimizer. It is becoming an AI task launcher: users pick a task flow, get a structured prompt, receive a model recommendation, and send it into ChatGPT, Claude, Gemini, Kimi, or other AI tools in one flow.

## 60-second demo flow
1. Open the Chrome side panel.
2. Choose a demo scenario:
   - Business email
   - Debug code
3. Show that the user starts from a real task instead of a blank box.
4. Click **Optimize task**.
5. Show the before/after comparison:
   - original request
   - improved task spec
   - why the prompt was improved
6. Point to the model recommendation card and explain why that model fits.
7. Click **Launch now** or **Launch** to fill the result into the target AI chat.
8. Close by showing that good prompts are saved into history as reusable assets.

## Three key product messages
- **Task-first**: users do not need to know how to write prompts from scratch.
- **Model-aware**: prompt.ai helps choose the right model for the task.
- **Execution-ready**: output is ready to launch, not just ready to read.

## Best live-demo scenarios
### 1. Business email
Input idea: ask a client to confirm a quotation quickly, while staying polite and firm.
Why it works: easy to understand, high visual contrast between rough input and improved prompt.

### 2. Debug code
Input idea: analyze an error, explain root cause, and give revised code.
Why it works: shows task structure, clarity, and value to technical users.

## Suggested judge-facing framing
If asked whether prompt tools will be commoditized, answer:

> Pure prompt polishing will commoditize. prompt.ai is moving beyond that into a task launcher layer: it helps users structure intent, choose the right model, and launch execution inside their real workflow.
